# Gazebo ROS 2 sample

A simple package to get a 3D lidar pointcloud from gazebo to ROS 2.
This repo makes use of the Gazebo ROS package to acheive the above goal.
REALIZED AN IMPORTANT LIMITATION GAZEBO BLOCK LASER IS NOT AVAILABLE IN ROS 2 ,
MAKING A PLUGIN TO ALLOW ROTATION OF THE LASER SCANNNER TO GET A NICE POINTCLOUD.
